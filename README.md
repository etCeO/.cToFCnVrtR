# TempSwitch 
### Celcius To Fahrenheit Converter

- This program is a lightweight Python-based temperature conversion utility that converts temperature values from degrees Celsius (°C) to degrees Fahrenheit (°F). It applies the standard linear transformation formula used in thermodynamics and meteorology:

°F=(°C×9/5)+32

The program is designed for accuracy, simplicity, and ease of integration into larger applications or scripts.
